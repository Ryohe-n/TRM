# 9.6.4 ACTMON Registers

Refer to "Reading Register Tables" in the Introduction chapter for the register table protocol as well as recommendations for accessing registers. The Base Addresses of the registers related to the Activity Monitor (ACTMON) are specified in the Address Map section of the Orin TRM.
- UNIT_ACTMON_GLB_CTRL_0
- Offset: 0x0
- Read/Write: R/W
- Parity Protection: N
- Shadow: N
- SCR Protection: 0
- Reset: 0x00000000 (0bxxxx,xxxx,xxxx,xxx0,xxxx,0000,0000,0000)

- Reset
