# 19.2 MHz or 38.4 MHz

UFS reference clock.
- PCIe Reference Clock Outputs
The HSIO PCIe controllers have one differential reference clock pair output per controller used as a reference clock for the endpoint associated with that controller. This is a 100 MHz output used for PCIe root mode and is driven out of the SoC by a separate pad and not the HSIO UPHY itself.
