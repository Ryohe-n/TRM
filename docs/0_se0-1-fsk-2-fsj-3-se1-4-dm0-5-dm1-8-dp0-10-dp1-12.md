# 0 = SE0 1 = FSK 2 = FSJ 3 = SE1 4 = DM0 5 = DM1 8 = DP0 10 = DP1 12 = NONE 13 = DMEDGE 14 = DPEDGE 15 = ANY 0x0 LINEVAL_WALK_EN:

- Perform Walk on USB line value wake up for UTMIP
0x0 LP_CFG_ENABLE:
- Used to enable some LP features (weak pd/pu ... etc)
0x0 WAKE_WALK_EN:
- Perform Walk on any chip wake up event for UTMIP
0x0 GPIO_WALK_EN:
- Perform Walk on associated GPIO event for UTMIP
