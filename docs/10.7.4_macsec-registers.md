# 10.7.4 MACsec Registers

- MACSEC_GCM_KEYTABLE_CONFIG_0
- GCM KeyTable Registers
- Indirect Addressing Access: Register controls for programming the Key tables GCM KeyTable
- Config Register
- Offset: 0x0
- Read/Write: R/W
- Parity Protection: YT194
- Shadow: N
- Secure: Trust Zone Protected
- SCR Protection: 0
- Reset: 0x00000000 (0b0xxx,xx00,xxxx,xxxx,xxxx,xxxx,xxx0,0000)
- Bit
- Reset
- Description
0x0 UPDATE:
- Key Table Configuration Update control
- SW has to trigger by writing Logic 1 to the UPDATE field after
programming DATA and other config fields. Poll for the value to become Logic0 to ensure the operation is complete. 0 = DONE :Update operation is done or No update operation is initiated 1 = START :Start the Update operation 0x0 CTLR_SEL:
- MACSEC Controller selection (Transmit or Receive)
