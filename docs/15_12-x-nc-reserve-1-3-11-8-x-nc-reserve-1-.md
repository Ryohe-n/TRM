In certain cases, the IP driver may implement diagnostics in SW. Errors detected by such SW diagnostics need to be reported to HSM. To avoid time consuming IPCs, IP SW can directly write an IP SW defined Error Code into the SW_ERR_CODE register and assert the SW Error. The IP Error Collator adds an additional error input with the below options: err_type = SW err_user - Adds err__user [31:0] as an input to the error collator for the err_x.
