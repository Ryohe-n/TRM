- Each bit of this field Indicates the Interrupt Status whether any GPIO
owned by a specific Guest OS resulted an Interrupt.
- Bit 0 is for VM1 (Guest OS0),
- Bit 1 is for VM2 (Guest OS1),
- Bit 2 is for VM3 (Guest OS2),
- Bit 3 is for VM4 (Guest OS3),
- Bit 4 is for VM5 (Guest OS4),
- Bit 5 is for VM6 (Guest OS5),
- Bit 6 is for VM7 (Guest OS6),
Bit 7 is for VM8 (Guest OS7). 0 = Interrupt not set. 1 = Interrupt set.
