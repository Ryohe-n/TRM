- Description
0x0 INITIAL_VAL:
- Start value for current Generator
0x0 ENABLE:
- When ENABLE goes from 0->1, the output level is unconditionally set to
INITIAL_VAL.
- The FSM goes into the WAITING state and starts processing EDGE registers
when TSC becomes equal or greater than the value programmed in TSC_START.
- When ENABLE is 0, the FSM is stopped in an IDLE state (WAITING and
- RUNNING both 0b)
