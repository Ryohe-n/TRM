# 31 :24 0x0 PSRQ3:

- Priorities Selected in the Receive Queue 3. This field decides the priorities
assigned to Rx Queue 3. All packets with priorities that match the values set in this field are routed to Rx Queue 3. For example, if PSRQ3[6, 3] are set, packets with USP field equal to 3 or 6 are routed to Rx Queue 3. The software must ensure that the content of this field is mutually exclusive to the PSRQ fields for other queues, that is, the same priority is not mapped to multiple Rx queues.
- When the Data Center Bridging feature is selected, this field also determines
the priorities to be included in the PFC packet sent to remote station when Rx Queue 3 crosses the flow control threshold settings.
