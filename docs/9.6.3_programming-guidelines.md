# 9.6.3 Programming Guidelines

@{Key=9.6.3; Title=Programming Guidelines; Body=This section provides an overview of the ACTMON register fields.
- Central ACTMON module contains a set of global registers which are common to all central
ACTMON Instances and some registers which are replicated for each ACTMON. The following are global registers, common to all central activity monitors.
- ACTMON Global Control
- ACTMON Global Interrupt Enable
- ACTMON Global Interrupt Status
The registers marked by XXX are replicated for each central ACTMON.
#### 9.6.3.1 Global Registers
ACTMONs only support 32-bit accesses. Byte accesses are not supported. Refer to ACTMON Registers for detailed descriptions of the global registers.
- ACTMON Global Control Register (ACTMON_MCGLB_CTRL_0)
- If the programmed value of SAMPLE_PERIOD is N,
the sample period duration would be (N+1) units (in mS/sample_tick or µs/sample_tick). This is the sample period for all ACTMON Instances within the central ACTMON module.
- If the SOURCE is selected as TICK, i.e., ACTMON reference clock,
sample period = (N+1) * (256 or 65356 depending on SAMPLE_TICK field) * unit, where unit
- ACTMON reference clock period.
- ACTMON Global Interrupt Enable Register (ACTMON_MC_GLB_INT_EN_0)
- For each Interrupt Enable bit: 1 for each ACTMON Instance (Instance 0 - 7) in the
ACTMON Wrapper.
- ACTMON Global Interrupt Status Register (ACTMON_MC_GLB_INT_STATUS_0)
- For each Interrupt Status bit: 1 for each ACTMON Instance (Instance 0 - 7) in the
ACTMON Wrapper.
#### 9.6.3.2 Central ACTMON Instance Registers
- The XXX in the ACTMON Instance register names is any of the ACTMON module names, i.e.,
MC_CPU, MC_ALL, TSA_A, TSA_B, TSA_C, DUAL_MCCIF_TSA, and TSA_RD_AVG_COUNT.

- Activity Monitor (ACTMON)
- The upper and lower watermarks of instantaneous and average count can be enabled
simultaneously by software.
- ACTMON_XXX Control Register (ACTMON_XXX_CTRL_0)
- ACTMON_XXX Interrupt Enable Register (ACTMON_XXX_INTR_ENABLE_0)
- Software needs to ensure that whenever the ACTMON is in the disabled mode, all the
corresponding Interrupt enables are set to disabled mode as well. In addition, software must ensure that the Interrupt status register is cleared before the ACTMON is re-enabled from a disabled state.
- ACTMON_XXX Interrupt Status Register (ACTMON_XXX_INTR_STATUS_0)
- ACTMON_XXX Upper Watermark Register (ACTMON_XXX_UPPER_WMARK_0)
This software-programmed register is the upper watermark for the ACTMON count value.
- ACTMON_XXX Lower Watermark Register (ACTMON_XXX_LOWER_WMARK_0)
This software-programmed register is the lower watermark for the ACTMON count value.
- ACTMON_XXX Average Upper Watermark Register (ACTMON_XXX_AVG_UPPER_WMARK_0)
This software-programmed register is the upper watermark for the ACTMON EWMA value.
- ACTMON_XXX Average Lower Watermark Register (ACTMON_XXX_AVG_LOWER_WMARK_0)
This software-programmed register is the lower watermark for the ACTMON EWMA value.
- ACTMON_XXX Init Average Value Register (ACTMON_XXX_INIT_AVG_0)
- This software-programmed register is the initial value of EWMA, corresponding to A0 in
the IIR filter formula.
- ACTMON_XXX Moving Average Register (ACTMON_XXX_AVG_COUNT_0)
- This is the EWMA count value. This is updated at end of sample period based on IIR filter
formula.
- ACTMON_XXX Monitor Counter (ACTMON_XXX_COUNT_0)
This is the ACTMON counter value. Note that this register is not updated at each clock. There is a separate counter which increments for each cycle the activity signal is ACTIVE.
- At end of a sample period, the value of this counter is scaled by the COUNT_WEIGHT and
then updated into this register. If read in middle of a sample window, the readback value is the old sample’s monitor value.
- ACTMON_XXX Count Weight Register (ACTMON_XXX_COUNT_WEIGHT_0)
- This register holds the COUNT_WEIGHT. The activity accumulated over the past sample
period is scaled by this weight and then updated into the Monitor Counter register.
- ACTMON_XXX Cumulative Count Register (ACTMON_XXX_CUMULATIVE_COUNT_0)

- ACTMON Registers
This register contains the cumulative counter value.}(@{Key=9.6.3; Title=Programming Guidelines; Body=This section provides an overview of the ACTMON register fields.
