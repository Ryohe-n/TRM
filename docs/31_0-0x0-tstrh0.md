0x0 TRGTBUSY0:
- PPS Target Time Register Busy The MAC sets this bit when the PPSCMD0 field in
the MAC_PPS_Control register is programmed to 010 or 011. Programming the
- PPSCMD0 field to 010 or 011 instructs the MAC to synchronize the Target Time
- Registers to the PTP clock domain. The MAC clears this bit after synchronizing
the Target Time Registers to the PTP clock domain The application must not update the Target Time Registers when this bit is read as 1. Otherwise, the synchronization of the previous programmed time gets corrupted.
