# 31 :0 0x0 DCMD_RESP:

- Direct Command Last Response
- This register contains the response of the command generated by the
last direct command (DCMD) task which was sent.
- CQE shall update this register when it receives the response for a DCMD
task. This register is considered valid only after bit 31 of CQTDBR register is cleared by CQE.
