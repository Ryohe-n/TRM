# 31 :16 0x0 L4DP21:

- Layer 4 Destination Port Number Field. When the L4PEN0 bit is reset and the
- L4DPM0 bit is set in the MAC_L3_L4_Control21 register, this field contains
the value to be matched with the TCP Destination Port Number field in the
- IPv4 or IPv6 packets. When the L4PEN0 and L4DPM0 bits are set in
- MAC_L3_L4_Control21 register, this field contains the value to be matched
with the UDP Destination Port Number field in the IPv4 or IPv6 packets.
