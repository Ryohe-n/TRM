##### 9.5.3.2.2 GPIO Common Control Registers
- This sub-section provides the description of GPIO Common Control registers that support
Virtualization and Security Control features under the control of Hypervisor. These registers are only accessible by the Hypervisor but not by the Guest OS's (VM's) because their address space is not visible to the Guest OS's (VM's).
