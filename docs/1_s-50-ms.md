# 1 µs 50 ms

- EQ interval between periodic events
- If the timer expires during single lane mode, then the controller should enable the remote
transmitter in order to train all lanes together in the same condition as normal high-speed mode. If the timer expires during L1 mode, then the controller is permitted to mark the periodic request as pending and perform the EQ event when exiting from L1.
- Abort
If the logic needs to enter a link state without high-speed data, it would be preferable to wait until it completes. However, the EQ logic supports receiving abort request if the logic is unable to wait due to other factors. However, it should wait for the DONE signals before proceeding to SLEEP state to ensure a clean abort sequence.
