# 7.8.3 Programming Guidelines

@{Key=7.8.3; Title=Programming Guidelines; Body=#### 7.8.3.1 Normal Operation (Enabling)
Program the CTRL_CHANNEL_SELECT field based on mono/stereo. Program the AODMIC_APB_FIFO_CTRL_THRESHOLD value. Program global enable by writing into AODMIC_ENABLE. 1. 2. 3.
#### 7.8.3.2 Disabling
Disable AODMIC_ENABLE by writing 0.
- Check the AODMIC_APB_FIFO_STATUS_DATA_COUNT; if non-zero. then program DMA to read
the data from FIFO and empty it.
- Check the AODMIC_STATUS_ENABLE_STATUS for 0. If AODMIC_STATUS_ENABLE_STATUS is
still '1' (due to pipe line data in internal controller) then repeat step 2 and step 3 *this step). 1. 2. 3.
#### 7.8.3.3 Optional
- Filter gains, AODMIC_DCR_FILTER_GAIN_0, AODMIC_LP_FILTER_GAIN_0,
- AODMIC_CORRECTION_FILTER_GAIN_0 can be programmed for volume control. The coefficients
are in the signed format with 23 LSB representing fractional part.}(@{Key=7.8.3; Title=Programming Guidelines; Body=#### 7.8.3.1 Normal Operation (Enabling)
