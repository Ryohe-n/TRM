- PMC IMPL Registers
- PMC_IMPL_SC7_STATUS_0
- If SW wants to check for SC7 residency, it should check for SC_STATE (this corresponds to
soc_wake_req). If SW wants to check for sc7 FSM idle-ness , it should check for SC7_FSM_BUSY SC7 FSM may be non-idle, but system may still not be in SC7, since true SC7 happens when all steps - assert SRAM_SLEEP, assert clamp, PWROFF happen. In those states FSM will be non-idle, but SC_STATE may still be NONE.
