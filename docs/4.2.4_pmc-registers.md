# 4.2.4 PMC Registers

Refer to "Reading Register Tables" in the Introduction chapter for the register table protocol as well as recommendations for accessing registers. All PMC configuration registers are implemented in the APB clock domain. Read/write access to these registers does not result in any wait state generation on the APB bus.
