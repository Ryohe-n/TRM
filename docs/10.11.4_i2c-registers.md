- I2C Registers
This register set is identical for all the I2C controllers.
- I2C_I2C_CNFG_0
- IC Controller Configuration Register
The I2C_CNFG register is used to configure: the number of bytes to be transmitted or received the slave device type (either a 7-bit device or a 10-bit device) enable mode to send Start-byte or not to select either a single slave transaction or two slave transactions enable mode to handle devices that do not generate an ACK.
