# 10.3.3 Programming Guidelines

@{Key=10.3.3; Title=Programming Guidelines; Body=#### 10.3.3.1 Fuse Programming
Refer to the Fuses chapter in this TRM. There are four shared ODM-specific fuse bits (i.e., board dependent calibration) in a mutually exclusively manner by enabling only one of the HSIO Complexes sharing the same UPHY Lanes.
- These four ODM bits are included in the fuse register FUSE_MPHY_ODM_CALIB_0. The two LSB's
are used by boot before accessing the BCT.
- MPHY_ODM_CALIB[1:0] of FUSE_MPHY_ODM_CALIB_0 adjusts M-PHY pad TX_DRV_AMP,
TX_DRV_POST, and RX_CTLE to compensate for different trace lengths.
