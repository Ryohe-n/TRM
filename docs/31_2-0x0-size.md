# 31 :2 0x0 SIZE:

- Size of the Address Range Firewall in bytes. Reset value of 0x00000
indicates firewall is disengaged. Bits [1:0] are tied off in design to 00 and writes to these bits are dropped.
- SW can assume these bits to be implemented and directly configure the
size in bytes Size[31:0] = 0x0 : Reset value. Illegal if ARF.WEN/ARF.REN = 1.
- Size[31:0] = 0x4 : bytes protected
- Size[31:0] = 0x8 : 8 bytes protected
- Size[31:0] = 0x10 : 16 bytes protected
- Size[31:0] = 0x20 : 32 bytes protected
...
- Size[31:0] = 0x8000000 :128MB protected
- Size[31:0] = 0x40000000 : 1GB protected
- Size[31:0] = 0x80000000 : 2GB protected
- Size[31:0] = 0xFFFFFFFC : 4GB-4Bytes protected
