There are two register groups defined for audio: one is for the FPCI interface to the system, and the other is for audio. The FPCI registers have 256 bytes (0x0~0xFC). It includes FPCI configuration registers, AZA Hardware debug registers, registers like OB FIFO monitor, OB FIFO watermark, etc. Registers can be accessed by: 1.
- NV_HDA_APB_PRI_CFG (or NV_ADDRESS_MAP_HDA_BASE+
- NV_HDA_APB_DFPCI_CFG_OFFSET) + offset (via PRI bus)
- NV_HDA_APB_DFPCI_CFG (or NV_ADDRESS_MAP_HDA_BASE) + offset (via FPCI bus)
2. HDA registers have 16 Kbytes (0x0000~0x3FFF) in BAR0 and it includes all HDA registers.
- HDA registers can be accessed by NV_HDACODEC_APB_BAR1_START + offset (via FPCI bus)
