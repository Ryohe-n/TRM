# 10.12.3 UART Registers

Refer to "Reading Register Tables" in the Introduction chapter for the register table protocol as well as recommendations for accessing registers. The UART is based on the 16550 industry standard Universal Asynchronous Receiver/Transmitter (UART), with enhancements to support auto-baud detection and End-of-Received Data timeout detection. The THR, RBR, and DLL registers all occupy the same address space.
- The Transmitter Holding Register (THR) is Write-Only, and can be accessed if the LSR.DLAB
bit is clear. The Receiver Buffer Register (RBR) is Read-Only, and can be accessed if the LSR.DLAB bit is clear.
- The Divisor Latch LSByte Register (DLL) is Read/Write and can be accessed if the LSR.DLAB
bit is set.
